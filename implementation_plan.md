# 明日方舟科幻 HUD 风格重构计划

本计划详细说明了如何根据视频中的视觉风格，将你的个人开发者数字空间门户进行重构，采用《明日方舟》科幻战术 HUD 风格重新设计主页、干员档案介绍、生活/技术分享以及项目经历页面。

## 视频站视觉风格总结

根据对屏幕录制视频的详细分析，该网站的核心视觉特征如下：
1. **配色方案**：以深色/黑灰色（`#0A0A0C`）为底色，以**电子青/蓝**（`#00f0ff`）、**警告橙/黄**、**荧光绿**（`#a6f626`）和纯白作为高亮和点缀色。
2. **HUD 战术元素**：随处可见坐标十字准星、虚线网格参考线、环形仪表盘、扫描动效、大写英文战术编码（如 `INDEX // 01`、`WORLD // 03` 等）以及包围边框。
3. **排版**：使用粗壮无衬线大写英文字体做主标题，配合等宽字体展示技术参数和段落信息。
4. **结构与交互**：
   - **主页**：左侧滑动浮现公告，中间为 3D 旋转概念模型。
   - **个人介绍**：罗德岛干员档案卡布局，左侧为技术战术指标与代号，右侧为精美立绘，底部为多卡片切换头像。
   - **技术与生活分享**：左侧为粒子星云 Canvas，点击右侧不同文章分类时，粒子星云可平滑变形组合成不同几何形态。
   - **项目经历**：三维等距（Isometric）桌面交互，悬停或点击不同的电子设备会亮起并展示对应的子版块（如画廊、视频等），配合卡片网格列表。

## 用户确认事项

> [!IMPORTANT]
> **资源与立绘要求**：
> - 为了在**干员介绍**中达到最佳的科幻立绘效果，我们需要一张高分辨率的个人形象照片或角色立绘。如果没有，我们将使用极具科技感的开发者剪影，配合流光边缘和网格叠加。
> - 我们将使用 Canvas 粒子系统开发“源石技艺”粒子动画，实现技术与生活分享页面的平滑粒子变形过渡效果。
>
> **等距交互桌面开发**：
> - 我们将使用高品质的 SVG 结合 CSS 3D 转换绘制等距开发桌面（显示器、机柜模型、全息投影、街机），实现点击亮起并弹出相应项目架构和日志的交互体验。

## 验证与测试方案

### 自动化测试
- 在 `frontend` 目录下运行 `npm run build`，确保所有组件的 JSX 语法、Tailwind 4 指令以及打包逻辑均编译成功，无任何 Error 或 Warning。
- 运行 `npm run lint` 验证代码规范。

### 手动功能验证
- 启动本地服务器 `npm run dev`：
  1. 测试 `HudHeader` 选项卡点击定位及滚动时的高亮同步。
  2. 测试 `OperatorProfile` 底部分页切换时数据和背景装饰的变化。
  3. 测试 `OriginiumSharing` 切换文章分类时，左侧 Canvas 粒子云的变形动画是否顺畅。
  4. 测试 `ProjectDesk` 桌面各电子设备的悬停发光与点击弹窗显示。
  5. 检查移动端自适应表现，确保所有 HUD 浮层与排版在小屏下优雅降级。
